## 长期架构（P3）

### Kernel 新模块
- kernel/wrapper（契约级 Traced 代理）
- kernel/command（L4 命令队列 + 状态机）
- kernel/webhook（出站 + HMAC + SSRF）
- kernel/reconcile（L3 Reconciler 模式）
- kernel/replay + kernel/rollback（投影重算 + 回滚元数据）
- runtime/scheduler（Cron + 分布式防重）

### 契约强化
- DURABLE-TYPE-01 + G-6 类型保护
- CONTRACT-BREAKING-01 / CONTRACT-CODEGEN-01 / CONTRACT-STUB-01
- C-AC7 jti / C-L6 contract ID

### 分层重整
- PR-A22 Cell ISP 拆分（Cell + CellLifecycle + CellMetadata）
- PR-A23 ER-ARCH-01 Subscriber Setup/Run 双阶段
- PR-A24 adapter layering 分层（剩余部分）

## 排期原则

- v1.1 按季度滚动，不做 "v1.1 sprint"
- 每条 🟠 trigger 项在触发前保持 watch，不预分配人
- PR-Axxx review 遗留按 lane 批量清理（Auth lane / Outbox lane / Governance lane）
- 新增 review findings 走 post-merge 聚合，不开即时补丁 PR
